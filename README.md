# Financial Metrics Dashboard

Dashboard de metricas financieras con frontend en React + TS y API mock en FastAPI.
Analiza el codigo con tu agente de IA para definir reglas de trabajo/contexto y construir un banco de memoria.

## Que debes hacer en este repo

- Identificar y proponer reglas de trabajo y de contexto para el proyecto usando un agente de IA.
- Generar y documentar un banco de memoria del proyecto para acelerar futuras iteraciones.

## Como levantar el proyecto

```bash
docker compose up --build
```

Frontend: http://localhost:5173
Backend: http://localhost:8000 (docs en /docs)
